# wordcloud

### Installing

Clone the repo

```
git clone https://github.com/moonfallmaple/wordcloud.git
```

Change into the *wordcloud* project directory

```
cd wordcloud
```

Install project dependencies

```
npm install
```

And start the application

```
npm start
```

The app will automatically launch in your browser.
